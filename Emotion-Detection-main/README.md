# Emotion-Detection

Emotion Detection web application built using Python and Flask.

This application analyzes user text and detects emotions such as **anger, disgust, fear, joy, and sadness** using **IBM Watson NLP**.

The system returns the **dominant emotion** detected from the input text.

## Technologies Used
- Python
- Flask
- IBM Watson NLP

## Features
- Detect emotions from text
- Returns emotion scores
- Displays dominant emotion
- Handles invalid or blank input
